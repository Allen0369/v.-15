Note: Keep all subfolders and app.py in one parent (project) folder

How to run:

1. Download Python (https://www.python.org/downloads/)
- If latest doesn't work, download v. 3.10.11 (https://www.python.org/downloads/release/python-31011/)

2. On CMD, type:
- cd (path of project folder)
- python app.py

3. On CMD, install (if missing):
- pip install Flask
- pip install scikit-learn
- pip install keras
- pip install pandas
- pip install numpy

4. On browser, type the URL displayed on CMD

5. To terminate program, click ctrl+c on CMD

6. For testing, use the datasets in #define - Master > UI / UX > sample datasets
